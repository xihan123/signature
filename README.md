# Signature

[![Deploy](https://github.com/xihan123/signature/actions/workflows/deploy.yml/badge.svg)](https://github.com/xihan123/signature/actions/workflows/deploy.yml)
[![GitHub Pages](https://img.shields.io/badge/GitHub-Pages-brightgreen)](https://xihan123.github.io/signature/)
[![License](https://img.shields.io/github/license/xihan123/signature)](LICENSE)

基于 Canvas 的手写签名工具，支持米字格辅助线，适用于电子签名场景。

**在线体验**：[GitHub Pages / EdgeOne Pages](https://signature.xihan.website/)

## 功能特性

- **手写绘制**：支持鼠标和触摸，HiDPI 适配
- **米字格**：可开关的米字格辅助线，行数列数可调
- **画笔设置**：颜色、粗细可调
- **撤销**：支持多步撤销（最多 60 步），Ctrl+Z 快捷键
- **导出**：导出为 PNG，支持透明/自定义背景色，Ctrl+S 快捷键
- **响应式**：适配移动端

## 使用

浏览器直接打开 `index.html` 即可使用，无需构建。
